# Auto Profit Bot / CarFlip CRM

Telegram-бот для обліку авто з США: покупка, доставка, розмитнення, ремонт, деталі, продаж старих запчастин, продаж авто, профіт, ROI, місячні звіти.

## Швидкий старт

1. Встанови Node.js.
2. Створи бота в Telegram через @BotFather і отримай BOT_TOKEN.
3. Створи PostgreSQL базу.
4. Скопіюй `.env.example` в `.env` і заповни дані.
5. Встанови залежності:

```bash
npm install
```

6. Створи таблиці:

```bash
npx prisma generate
npx prisma migrate dev --name init
```

7. Запусти бота:

```bash
npm run dev
```

## Основні команди

- /start — головне меню
- /addcar — додати авто
- /cars — список авто
- /calculator — калькулятор потенційного профіту
- /analytics — загальна аналітика
- /month — звіт за місяць

## MVP функції

- Додавання авто
- Додавання витрат
- Додавання ремонту
- Додавання деталей
- Продаж старих деталей
- Продаж авто
- Автоматичний розрахунок собівартості
- Розрахунок чистого профіту і ROI
- Місячна аналітика
